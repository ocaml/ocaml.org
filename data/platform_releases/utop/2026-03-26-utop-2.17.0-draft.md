We're happy to announce the release of utop 2.17.0.

This release adds support for OCaml 5.5 ([#510](https://github.com/ocaml-community/utop/pull/510)) and decouples `utop.el` from `tuareg-mode` ([#511](https://github.com/ocaml-community/utop/pull/511)), so that other OCaml major modes for Emacs (such as [neocaml](https://github.com/bbatsov/neocaml)) can integrate with it via the new `utop-mode-compat-alist`; `tuareg` is now loaded lazily, and existing tuareg, caml, typerex, and reason users are unaffected.

For more details, see the full changelog below.
