We're happy to announce the release of Merlin 5.7.1-504, a patch release for the OCaml 5.4 series. This release brings:

- A fix for a typer cache invalidation bug ([#2062](https://github.com/ocaml/merlin/pull/2062)) where Merlin would not pick up newly created `.cmi` files appearing in the build path, for instance when Dune built a new module.
- A change to the on-disk index format ([#2051](https://github.com/ocaml/merlin/pull/2051)) so that merged indexes can point to existing index files, resulting in significantly smaller index files and fewer write operations.

For more details, see the full changelog below.
