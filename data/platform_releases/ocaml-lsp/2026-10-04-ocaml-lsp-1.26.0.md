changelog: |
    ## Features
    - Add custom [`ocamllsp/refactorExtract`](https://github.com/ocaml/ocaml-lsp/blob/main/ocaml-lsp-server/docs/ocamllsp/refactorExtract.md) request ([#1545](https://github.com/ocaml/ocaml-lsp/pull/1545))
    - Add `destruct` custom request ([#1583](https://github.com/ocaml/ocaml-lsp/pull/1583))
    - Add `locate_types` custom request ([#1584](https://github.com/ocaml/ocaml-lsp/pull/1584))
    - Add support for the `textDocument/rangeFormatting` request ([#1591](https://github.com/ocaml/ocaml-lsp/pull/1591))
    - Allow to configure merlin using build systems other than dune.
        If `$OCAMLLSP_PROJECT_BUILD_SYSTEM` environment variable is set then OCaml
        LSP will execute `$OCAMLLSP_PROJECT_BUILD_SYSTEM ocaml-merlin` command to get
        merlin configuration for a module. It is expected the command to communicate
        using `merlin-dot-protocol`.
        If `$OCAMLLSP_PROJECT_ROOT` environment variable is set then OCaml LSP will
        use it as a project root directory.

    ## Fixes
    - Enable "hover" on more locations, notably class and object related ([#1599](https://github.com/ocaml/ocaml-lsp/pull/1599))
---

This new release of OCaml-LSP brings several new features and improvements, including new custom requests, support for range formatting, and enhanced hover capabilities.

See the related [announcement on the Discuss forums](https://discuss.ocaml.org/t/ann-new-release-of-ocaml-lsp-1-26-0-merlin-5-7-0-504/17978).
