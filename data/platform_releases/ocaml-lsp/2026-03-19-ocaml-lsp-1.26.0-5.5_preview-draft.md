<h2>Features</h2>
<ul>
<li>
<p>Add <code>destruct</code> custom request (<a href="https://github.com/ocaml/ocaml-lsp/pull/1583" class="issue-link js-issue-link" data-error-text="Failed to load title" data-id="3833778556" data-permission-text="Title is private" data-url="https://github.com/ocaml/ocaml-lsp/issues/1583" data-hovercard-type="pull_request" data-hovercard-url="/ocaml/ocaml-lsp/pull/1583/hovercard">#1583</a>)</p>
</li>
<li>
<p>Allow to configure merlin using build systems other than dune.</p>
<p>If <code>$OCAMLLSP_PROJECT_BUILD_SYSTEM</code> environment variable is set then OCaml<br>
LSP will execute <code>$OCAMLLSP_PROJECT_BUILD_SYSTEM ocaml-merlin</code> command to get<br>
merlin configuration for a module. It is expected the command to communicate<br>
using <code>merlin-dot-protocol</code>.</p>
<p>If <code>$OCAMLLSP_PROJECT_ROOT</code> environment variable is set then OCaml LSP will<br>
use it as a project root directory.</p>
</li>
</ul>
