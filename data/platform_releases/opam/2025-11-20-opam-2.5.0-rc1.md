Hi everyone,

We are happy to announce the first release candidate of opam `2.5.0`, which boasts a grand total of zero (0) changes compared to `2.5.0~beta1`!

Try the new opam 2.5.0 release candidate:

The upgrade instructions are unchanged:

For Unix systems

bash -c "sh <(curl -fsSL [https://opam.ocaml.org/install.sh](https://opam.ocaml.org/install.sh)) --version 2.5.0~rc1"

or from PowerShell for Windows systems

Invoke-Expression "& { $(Invoke-RestMethod [https://opam.ocaml.org/install.ps1](https://opam.ocaml.org/install.ps1)) } -Version 2.5.0~rc1"

Please report any issues to the bug-tracker.
